<section>
  <h2>二、每日行程重點</h2>

  <p><strong>Day 1｜抵達沖繩・海炎祭海上花火</strong><br>
  抵達沖繩後前往會場，欣賞沖繩年度盛事「琉球海炎祭」，結合音樂與大型海上花火，氣氛震撼，是當地極具代表性的活動。</p>

  <p><strong>Day 2｜美麗海水族館・古宇利島・美食饗宴</strong><br>
  參觀世界級沖繩美麗海水族館，欣賞鯨鯊與黑潮之海；前往有「戀之島」之稱的古宇利島。午餐安排水族館自助餐，晚餐享用沖繩名物阿古豬料理。</p>

  <p><strong>Day 3｜在地漁市場・瀨長島・PARCO CITY</strong><br>
  走訪在地漁市場，體驗沖繩日常生活；前往瀨長島 Umikaji Terrace，感受純白建築與海景風光；下午至 PARCO CITY 購物中心自由選購。</p>

  <p><strong>Day 4｜玉泉洞・波上宮・美國村</strong><br>
  參觀玉泉洞鐘乳石洞，欣賞自然奇景；前往沖繩代表性神社波上宮參拜；傍晚至美國村欣賞異國風街景與夜色。</p>

  <p><strong>Day 5｜國際通自由行・返程</strong><br>
  國際通自由活動，選購伴手禮與特色商品，隨後前往機場返程。</p>
</section>
